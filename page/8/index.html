Concurrent-AMAT, C-AMAT

并发式平均存储访问时间模型

APC

侧重于测量方法的研究， 给出了并发存储的测量方法和尺度。

在 APC 中， 周期是存储活动周期 (memory
active cycle)， 不是通用的 CPU 周期，所以 APC 也叫 APMAC（ 存储活动周期平均访问数， access per
memory active cycle）。

同时 APC 采用重叠 (overlapping) 的访存时间统计方法 ： 在有两个或多个存储访
问同时进行时， 周期只增加一次。
